    Permission Justification

    This extension uses:

    • storage permission to keep interceptor on/off status and options persistent.
    • webRequest permission to redirect or match outgoing HTTP requests.
    • webRequestBlocking permission to block image requests with simple string matches.
    • http://*/* and https://*/* host permissions to filter and redirect/block HTTP requests.
    • notifications permission to inform users on updates, like configuration options' updates.

    Other

    • This extension do not use sync storage area, so your extension data will not be synced across the browsers you logged into!
    • Other extensions' redirecting/blocking behaviour may conflict with this extension. (like ad/tracker blockers)
    • Never keep confidential data (like passwords) inside redirection or match URLs.
